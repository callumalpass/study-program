# Topic 1 Practice Exercises

## Conceptual Questions

1. **Definition Review**: Explain the key concepts covered in this topic in your own words.

2. **Comparison**: Compare and contrast the main algorithms/approaches discussed.

3. **Applications**: Describe 3 real-world applications of the techniques from this topic.

## Problem Solving

4. **Algorithm Implementation**: Implement the main algorithm from this topic in your preferred language.

5. **Complexity Analysis**: Analyze the time and space complexity of your implementation.

6. **Optimization Challenge**: Can you improve the performance for special cases?

## Theoretical Exercises

7. **Proof Construction**: Prove the correctness of one of the key algorithms.

8. **Lower Bound**: Argue why certain problems in this topic cannot be solved faster.

9. **Reduction Design**: Create a reduction from one problem to another.

## Coding Challenges

10. **Test Suite**: Write comprehensive unit tests for your implementations.

11. **Benchmarking**: Compare performance across different input sizes and types.

12. **Variant Implementation**: Implement a variant or extension of the main algorithm.

## Advanced Problems

13. **Research Question**: Investigate current research directions related to this topic.

14. **Optimization**: Apply the techniques to solve a practical optimization problem.

15. **Integration**: Combine concepts from multiple topics to solve a complex problem.
